# HDMI-to-USB-C-DisplayPort-
This project designs an HDMI to USB-C DisplayPort Adapter using KiCad 9.0.7. The system converts HDMI TMDS signals into DisplayPort signals using the STDP2600AD bridge IC and supports USB-C DisplayPort Alternate Mode communication through TUSB1046A and FUSB302 ICs for modern USB-C display connectivity.

You will find the following:

- **Board Schematic:** A detailed circuit design of the HDMI-to-USB-C-DisplayPort.
- **PCB Layout:** The physical design of the printed circuit board.
- **PCB Routing:** The connections and pathways for signals and power on the PCB.


### Tools Used

The electronic design and development were done using **KiCad**, an open-source EDA software.
The version used for this project is **KiCad 9.0.7**.

#### The footprints of all Resistors, Capacitors:
`0402_1005Metric_Pad0.74x0.62mm_HandSolder`

All the files are attached in the repository folder.
