# HDMI-to-USB-C-DisplayPort-
This project designs an HDMI to USB-C DisplayPort Adapter using KiCad 9.0.7. The system converts HDMI TMDS signals into DisplayPort signals using the STDP2600AD bridge IC and supports USB-C DisplayPort Alternate Mode communication through TUSB1046A and FUSB302 ICs for modern USB-C display connectivity.
